# Cine-proyecto

Este proyecto es una plataforma de gestión de películas que permite visualizar una cartelera dinámica, ver detalles específicos de cada pelicula y realiza simulaciones de renta calcualando el precio por los dias de renta.

## 🚀 Funcionalidades Implementadas
- **Cartelera Dinámica:** Obtención de información a través de AJAX desde un archivo en formato JSON.
- **Lógica de Precios:** Distinción automática entre Estrenos y En Cartelera basada en fechas menores a 30 días.
- **Spinner de Carga:** Simulación de un tiempo de carga de 5 segundos.
- **Sistema de Renta:** Formulario operativo que calcula el costo total según el número de días y películas seleccionadas.
- **Página de Detalle:** Visualización interactiva de la sinopsis y los tráilers de YouTube utilizando parámetros de URL.
- **Diseño Moderno:** Implementación del Modo Oscuro, tipografías de Google (Poppins) y efectos de hover aplicados con CSS.

## 📂 Estructura del Proyecto
- `/index.html`: Página principal de la cartelera.
- `/pages/`: Contiene detalle.html y contactos.html y una nueva clase llamada renta.html.
- `/data/`: Archivo de peliculas.json con la base de datos.
- `/img/`: Posters de las películas en cartelera y estreno.
- `/js/`: Lógica principal (app.js).
